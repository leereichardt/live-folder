# Live Folder

Track your github pull requests automatically and have them ready for review in your browser's bookmarks.

## Build

Build for Firefox:
```bash
bun install
bun run zip:firefox
```

Build for Chrome:
```bash
bun install
bun run zip
```

## Development

Run the development server:
```bash
bun install
bun run dev
```

Run the development server for Firefox:
```bash
bun install
bun run dev:firefox
```
